## 接入 Arcadia

**Aetherpunk** 是一款以 AI 叙事为核心、本地规则系统托底的网页跑团 RPG。

没有固定选项。没有写好的结局。没有人在等你做出“正确选择”。

你进入的是 **阿卡迪亚**，一座会回应、会记忆、会追债的城市。你在这里做的每一件事，都将成为它的一部分，无论你愿不愿意。

## 世界 · 阿卡迪亚

城市上空，悬浮着阿卡夏的传导水晶。那东西又大又亮，把蓝白冷光投进积水的路面，也把你的每一次施法印进它的中央服务器。

哥特尖塔、希腊式殿堂、冷酷的精英商业建筑，三种时代的遗址被挤在同一座城里，用魔法光辉的管线和天轨交通网络缝在一起，像一个正在运转的矛盾体。这里的居民大多混合了多种血统：尖耳的精灵裔、带翅膀的翼人后代、局部生着龙鳞的混血者，以及无数无法被归类的人。纯血早就成了传说，阿卡迪亚不在乎你从哪里来，只在乎你的社会信用评级。

**奥域网 The Arcanet** 是让这一切运转的神经系统。它高效、稳定、便宜，把施法、认证、支付、通讯和监控打包成一个你很难拒绝的订阅服务。阿卡夏说，这是便利。合唱书的人说，这是最华丽、最舒适、也最坚不可摧的囚笼。

### The Suit · 公司秩序

> “已授权。已认证。已记录。”

摩天大楼林立，全息广告牌覆盖天际，空中交通井然有序。奥域网的核心办公区在这里，合规是规则，效率是神，预测是武器。

### The Street · 街头网络

> “没有总部。没有领袖。没有签名。”

复杂的立体街区，霓虹灯、涂鸦、地下俱乐部，以及写在墙角的合唱书暗号。所有不被主流接纳之人都在这里，和他们带来的东西一起生存。

### The Academy · 学院传统

> “荣誉在先，权限在后，门槛永远存在。”

古老建筑散发历史的厚重感。魔法最高学府、古老贵族庄园、宏伟教堂，以及一套将传统魔法和现代权限系统缝在一起的礼仪体系。优雅是武器，阶层从未消失。

| **Spire** | 学院礼仪、古典研究、贵族纹章 | 所有人都在评估你的血统和权限等级。 |

## 历史还在呼吸

以太流被发现时，法师们以为一个真正互联互通的时代即将来临。然后，阿卡夏来了。

奥域网诞生的第 15 年，《净化法案》出台，禁止所有“未经认证”的魔法知识传播，所有法术必须经过奥域网审查。表面理由是防止危险魔法扩散。实际原因是垄断，是消灭竞争，是全面监控。

**合唱书 Codex Chorus** 没有消失。它转入地下，寄生在奥域网自己的以太流里，像暗流藏在河底，在阿卡夏以为它死去之后继续呼吸。

- **10 年前**，“清洗日”让 Chorus 大规模断线。整整 72 小时，许多节点永久沉默，没有任何人解释发生了什么。
- **5 年前**，某件事证明地下网络无法被彻底消灭。
- **3 年前**，社会信用算法源码泄露，证明阿卡夏在用魔法预测“危险人物”。Chorus 用户数量在随后几个月暴增了五倍。

这些日期活在 NPC 的创伤里，活在委托的动机里，也活在某些对话里一闪而过的沉默里。

## 魔法 · 以太技术

这个世界的魔法有版本号，有兼容性问题，有被强制更新的风险，也有被吊销授权的可能。

| 概念 | 说明 |
| --- | --- |
| **以太流 Aether** | 遍布世界的魔法能量场。意识与信息在其中流动，也在其中腐烂；偶尔你会听见它的低频共振，像千万人同时低语。 |
| **终端魔法 Terminal Magic** | 咒文移植为 App、插件、黑市模块和硬件负载。用个人终端对着肉串打出“火花术”，或者用改装模块绕过奥域网授权审查，都叫终端魔法。 |
| **奥域网 The Arcanet** | 阿卡夏的中心化魔法基础设施。城市上空那颗传导水晶就是它的中央服务器，稳定、高效、全面监控。 |
| **合唱书 Codex Chorus** | 无总部、无领袖的地下知识网络，靠节点共识、幽灵协议和活性记忆生存。它存储的法术有版本历史、演化树和用户评分，像活着的开源项目。 |
| **以太天气** | 细雨、大雾、雷暴、以太雨、水晶雾。天气改变场景气压，也改变你的选择空间。 |
| **回音 The Echo** | 偶尔在使用合唱书时出现的奇异声音，像千万人同时低语；没有人确定那是什么。 |

终端噪声、浮空水晶的冷光、全息广告牌的闪烁、社会信用警报、trace 暴露、以太低频、记忆错位，现代技术与神秘魔法彻底重叠之后，那条缝已经无法分辨。

## 玩法 · 成为 Runner

### 一、创建你的 Runner

从血缘、派系、出身、属性、终端、装备开始，回答叙事问卷，建立一个有负伤记录、有过去关系、有资源缺口的人。带着这些进入阿卡迪亚。

### 二、接受任务简报

每个委托都有目标、联系人、已知风险与可能后果。选择结构化委托，或者直接走进城市某个区域，让当前的角色状态决定城市怎么回应你。

### 三、组建队伍

选择主 Runner 与最多两名同行者。队友有装备、库存、立场与价值判断。某些门，一个人根本敲不开；某些交易，因为亲密度或派系关系会变成完全不同的价格。

### 四、用自然语言行动

潜入、交涉、威胁、调查、撤退、购买服务、请求 NPC 帮忙、转移物品，或者开战。描述你想做的，AI 负责叙述世界如何回应，本地系统负责保存和校验每一个关键结果。

你的行动不会被下一轮对话覆盖。

### 五、世界会继续运转

你离开之后，阿卡迪亚不会等你。时间推进，天气变化，NPC 积累好感与事件，地点留下记忆，任务归档。

你上次留下的未了事宜，正在变成某人的动机。

## 系统 · 记录一切

- **叙事有据可查。** 任务目标、角色资源、战斗、交易、NPC 关系与任务结算由本地系统追踪。不靠记忆力，不靠重新描述。
- **世界记住你做过什么。** NPC 档案、地点记忆、天气历史与任务存档把每一次选择堆积成长期故事。你帮过的人会记得，你得罪的人也会记得。
- **NPC 是真实存在的。** 任务角色与现场生成 NPC 都能进入档案，保留性格、人称、血缘关系、历史事件与战斗记录。他们不会在任务结束后消失，除非你让他们消失。
- **阿卡迪亚是可互动的舞台。** 区域气压、以太天气、派系动态、社会信用状态、合唱书节点与奥域网监控，是行动约束，也是机会来源。
- **Runner 有长期成长。** 属性提升、Specialty 解锁、终端 App 扩展、装备挂载、负伤积累与任务履历，让角色慢慢形成只属于自己的打法，和在城市里的位置。

## 适合你的时候

- 你喜欢赛博朋克、都市奇幻、魔法工业化与地下网络叙事。
- 你想用自然语言跑团，但不希望角色资产、关系和状态被随意抹掉。
- 你想把 AI 当真正的叙事主持人。
- 你想做长线角色，把 NPC 档案和城市记忆慢慢建成一个属于你的故事数据库。

## 本地启动

```powershell
npm install
npm run dev
```

打开本地地址后，从首页选择 **New Runner** 创建角色，进入 **New Mission** 选择任务简报，最后通过 **Continue** 接入当前任务。
